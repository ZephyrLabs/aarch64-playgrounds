# Template program for OpenCL experiments:
example template script for testing OpenCL on the Mali GPU, written in both C and C++.

utilizes the full profile OpenCL v2.1 spec.

### for C program:
full source here: [template/c](c)

``` 
cd c
make
./main
```

### for C++ program:

full source here: [template/c++](c++)
```
cd c++
make
./main
```
